Example: `<tool>{"tool_name": "edit", "path": "src/app.py", "edits": [{"oldText": "…", "newText": "…"}]}</tool>`
- `bash` — runs a real shell command on my machine: `command`, plus an optional `timeout` in seconds if you want one (there's no default timeout otherwise). Use this for listing directories, finding files, and searching content too (`ls`, `find`, `grep` as actual shell commands).
